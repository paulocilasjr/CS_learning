# CS_learning

`Terminal Quest` is a terminal-only tutorial game for kids. It teaches command-line basics, files and folders, and beginner Python through 150 small missions.

## Run It

```bash
python main.py
```

The game opens its own safe practice shell. It does not touch your real files.

## Helpful Options

```bash
python main.py --reset
python main.py --no-save
python main.py --start-task 106
```

- `--reset` starts from mission 1 and clears saved progress.
- `--no-save` runs without writing progress to disk.
- `--start-task` is handy for jumping to a lesson while testing.

## In-Game Commands

- `hint` shows a clue for the current mission.
- `repeat` shows the mission again.
- `progress` shows mission progress and stars.
- `reset` resets the current mission room.
- `exit` leaves the game.
